
# RecruitKit – AI Job Description Generator

RecruitKit is a lightweight web app that helps staffing agencies generate professional, inclusive job descriptions in seconds.  
It focuses on speed, consistency, and best-practice wording — without replacing recruiters or existing ATS tools.

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- Clerk (Authentication)
- Stripe (Subscriptions)
- OpenAI (JD generation)
- PlanetScale (MySQL)
- Prisma (ORM)
- Tailwind CSS
- Vercel (Hosting)

---

## Prerequisites
- Node.js 18+
- GitHub account
- Free accounts with:
  - Vercel
  - PlanetScale
  - Clerk
  - Stripe
  - OpenAI
